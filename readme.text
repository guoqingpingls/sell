vue核心思想
1.数据驱动
    DOM是数据的一种自然映射 数据驱动DOM变化
    数据改变驱动视图自动更新
2. 组件化
    扩展html元素，封装可重用的代码
    设计原则： 页面上每个独立的可视/可交互区域视为一个组件
              每个额组件对应一个工程目录，组件所需要的各种资源在这个目录下就近交互
              页面不过是组件的容器，组件可以嵌套自由组合形成完整的页面
